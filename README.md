# WIP - Python Task Tracking App

Planning workspace for a Python 3.11.2 work task tracking application with:
- task lifecycle dates (received, due, closed)
- recurring task automation
- timestamped task notes history
- todo milestone tracking
- calendar-based planning views

## Documents
- `app_overview.md` - product goals, workflow, and relationship baseline from Access
- `features.md` - MVP and post-MVP feature set
- `tech_decisions.md` - architecture and technology choices
- `questions.md` - open decisions to finalize before implementation
- `todo.md` - phased implementation checklist
